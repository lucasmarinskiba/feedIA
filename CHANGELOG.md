# Changelog — FeedIA

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- CI/CD modular with GitHub Actions.
- Docker multi-stage build for workers.
- Supabase migrations managed via CLI.
- Post-deploy smoke tests.
- Sentry and structured logging observability.

## [1.0.0] - 2026-05-14

### Added

- Sprint 11 complete: self-improvement engine, feedback collector, strategy tuner, AR creator.
- Multi-agent CMO for Instagram/TikTok.
- Vercel frontend with security headers and CSP.
- Rate limiting by credits and plan.
- Supabase RLS schema for multi-tenancy.
- Redis cache and BullMQ workers.
