# FeedIA Pinterest Extraction Plan

## Mission
Extract narrative patterns, design systems, copy strategies, tools from your Pinterest pins.
Train FeedIA brain to generate carousels with:
- Coherent narrative (3-10 slides, no filler)
- Retention curve (each slide = reason to swipe)
- Design language (fonts, colors, backgrounds)
- Copy mechanics (hooks, value escalation, proof, CTA)

---

## Carousel Narrative Rules (from your pins)

### Slide Count Decision Tree
- **3-4 slides**: Micro-content. Single topic. No objections. CTA-heavy.
- **5-6 slides**: Classic. Hook → Value → Proof → CTA. Straightforward.
- **7-8 slides**: Deep dive. Hook → Curiosity → Multiple values → Objection handler → Proof → CTA.
- **9-10 slides**: Only if narrative DEMANDS it. Must have clear arc. Otherwise = filler.

**Rule**: If story doesn't need slide → DELETE. Retention drops 10-15% per weak slide.

### Narrative Structure Per Slide

| Slide # | Role | Purpose | Copy Pattern | Design | Retention Hook |
|---------|------|---------|--------------|--------|-----------------|
| 1 | Hook | Stop scroll | Question OR Controversy OR Social proof | Hero image full-bleed | Pattern interrupt. "Wait..." |
| 2 | Curiosity | Build tension | Contrast (myth vs reality) OR revelation | Gradient bold colors | Mystery. "But here's..." |
| 3+ | Value | Deliver lesson | First/second/third tip with example | Image + text overlay | Quick win. "Today..." |
| n-2 | Objection | Address "but what about" | Common myth + reframe | Testimonial style | Remove hesitation |
| n-1 | Proof | Credibility | Case study OR stat OR before-after | Solid accent background | "Others got..." |
| n | CTA | Action | Direct ask + benefit + deadline | Clean, minimal | Link/DM/Follow |

## What to Extract from Each Pinterest Pin

### 1. NARRATIVE ANALYSIS
When you view each pin, ask:

**Hook (Slide 1):**
- [ ] What's the pattern interrupt? (Question / Controversy / Social proof / Curiosity)
- [ ] Does first slide make you NEED next slide?
- [ ] Energy level? (Calm / Medium / High)
- [ ] Design: Full-bleed image? Gradient? Solid?

**Value Slides (Slides 2-7):**
- [ ] How many value points? (Don't count, quality > quantity)
- [ ] Format progression? (Headline-heavy → Image-heavy → Data-heavy)
- [ ] Each slide offer quick win? (Implementable today?)
- [ ] Objection handled? (Usually slide 5-6)

**Proof (Before Last):**
- [ ] Social proof? (Testimonial / Stat / Case study / Before-after)
- [ ] Makes promise believable?
- [ ] Design: Testimonial box? Stat highlight? Hero image?

**CTA (Last Slide):**
- [ ] Clear ask? (Follow / Link in bio / DM / Save / Share)
- [ ] Urgency or benefit? ("Last 5 spots" or "Worth $500")
- [ ] Friction removed? (One click away)

**Retention Questions:**
- [ ] Would you swipe to slide 2? Why?
- [ ] Would you swipe to slide 3? 
- [ ] At what slide would you stop swiping? (Weak narrative = early exit)

---

### 2. DESIGN ANALYSIS
Extract visual system:

**Typography:**
- [ ] Headline font: (Poppins? Playfair? Montserrat?)
- [ ] Headline size: (Estimate: 28px? 36px? 48px?)
- [ ] Body font: (Inter? Open Sans? Lora?)
- [ ] Body size: (14px? 16px? 18px?)
- [ ] Script/accent font: (Any decorative?)
- [ ] Font pairing feeling: (Elegant / Playful / Professional / Bold)

**Colors:**
- [ ] Primary color: (Hex if possible, else description)
- [ ] Secondary color: (Complementary or contrast?)
- [ ] Background colors: (White / Dark / Image / Gradient?)
- [ ] Text colors: (White on dark? Dark on light? High contrast?)
- [ ] Mood: (Warm / Cool / Neutral / Bold / Subtle)

**Layout & Elements:**
- [ ] Text placement: (Overlay on image? Side? Bottom? Center?)
- [ ] Text alignment: (Left / Center / Right?)
- [ ] Safe zone: (Text avoids edges?)
- [ ] Background style: (Image / Gradient / Solid / Pattern)
- [ ] Visual hierarchy: (Headline dominant? Data prominent? Image hero?)

**Visual Elements:**
- [ ] Icons used? (What kind? Outline or filled?)
- [ ] Illustrations or photos?
- [ ] Mockups? (Product mockups, device mockups, etc)
- [ ] Graphic elements? (Lines, shapes, decorative)
- [ ] Cross-slide continuity? (Does image flow to next slide?)

---

### 3. COPY STRATEGY ANALYSIS

**Hook (Slide 1):**
- [ ] Hook type: (Question / Controversial / Social proof / Curiosity / Fear / Benefit)
- [ ] Word count: (Short + punchy or longer setup?)
- [ ] Urgency level: (Immediate / Someday / Never)
- [ ] Emotion triggered: (Fear / Hope / Joy / Anger / Curiosity)

**Value Slides:**
- [ ] Copy technique: (Listicle / Before-after / Story / How-to / Myth-busting)
- [ ] Depth: (Actionable tip? Theory? Principle?)
- [ ] Tone shift per slide? (Consistent or varied?)
- [ ] Escalation pattern? (Basic → Intermediate → Advanced? Or same level?)

**CTA:**
- [ ] CTA type: (Follow / Link / DM / Save / Share / Tag)
- [ ] Benefit stated? ("For more tips" or "Link in bio")
- [ ] Scarcity/Urgency? ("Last 3 spots" or "Closes Friday")
- [ ] Friction level: (Easy one-click? Or complex requirement?)

---

### 4. TOOLS & RESOURCES MENTIONED

- [ ] Design tools: (Canva? Figma? Photoshop?)
- [ ] Video editing: (CapCut? Adobe Premiere? InShot?)
- [ ] Image sources: (Unsplash? Pexels? Adobe Stock?)

### 5. CONTENT COHERENCE CHECKLIST

For each pin, rate 1-5:

- [ ] **Narrative Clarity** (1-5): Easy to follow story?
- [ ] **Value Density** (1-5): Each slide adds new info?
- [ ] **Visual Cohesion** (1-5): Designs feel like same brand?
- [ ] **Copy Tone** (1-5): Consistent voice throughout?
- [ ] **Retention Curve** (1-5): Would you swipe all slides?
- [ ] **CTA Clarity** (1-5): Know exactly what to do?

## Example Pin Analysis (Template)

```
PIN: https://pin.it/6v5npgCT2
TITLE: "10 Mistakes You're Making"

NARRATIVE:
- Slide count: 10 (evaluate: necessary or filler?)
- Hook: "STOP. You're probably making these 10 mistakes right now"
- Pattern interrupt: ✅ Fear (worry you're doing it wrong)
- Value escalation: Slide 2 = basic mistake, Slide 5 = intermediate, Slide 8 = advanced
- Objection handled: Slide 7 ("But what if I've been doing this for years?")
- Proof: Slide 9 (stat: "87% of people make mistake #3")
- CTA: "Follow for more tips" (low friction, clear ask)
- Retention estimate: 75% (good but loses ~25% by slide 10)

DESIGN:
- Headline font: Poppins Bold
- Headline size: 40px
- Body font: Inter Regular
- Body size: 16px
- Primary color: #E91E8C (hot pink)
- Secondary: #00D9FF (cyan)
- Background: Photographic images (lifestyle/product shots)
- Text placement: Centered overlay with semi-transparent dark background
- Safe zones: 20% margins on all sides
- Visual hierarchy: Headline 40%, Image 50%, CTA 10%

COPY STRATEGY:
- Hook type: Numbered listicle (taps into curiosity + fear of missing out)
- Copy technique: "Here's what most people get wrong" (relatable failure)
- Escalation: Mistake #1 (obvious) → Mistake #5 (subtle) → Mistake #10 (advanced)
- Tone: Conversational, slightly sarcastic ("You probably do this without realizing")
- CTA: "Follow for more tips 🔥" (social proof + low commitment)

TOOLS MENTIONED:
- Canva (design)
- Adobe Illustrator (icons)
- Unsplash (images)
- Google Fonts (typography)

COHERENCE SCORES:
- Narrative: 4/5 (great flow, maybe 1-2 slides could be removed)
- Visual: 5/5 (consistent design language)
- Copy: 4/5 (clear but slightly repetitive by slide 8)
- Retention: 3/5 (loses audience by slide 9-10)

FEEDIA LEARNINGS:
→ Use 7-8 slides for "mistakes" format, not 10
→ Poppins + Inter pairing works well
→ Pink + Cyan palette = high engagement (test in similar industries)
→ Numbered listicles perform better than unnumbered
→ End at slide 8 with CTA (don't extend to 10)
```
