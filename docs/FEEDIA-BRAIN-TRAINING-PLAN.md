# FeedIA Brain Training Plan - Pinterest Research

## Mission
Train FeedIA brain from 50 Pinterest pins to generate viral carousels + videos with proven design patterns, copy strategies, typography, color systems.

## Pinterest Patterns Extracted

### Visual Design System

**Color Palettes (most common):**
1. **Dark Premium**: #1A1A1A (bg) + #E6D5B8 (gold) + #FFFFFF (text)
   - Used by: Premium products, education, professional services
   - Mood: Luxury, trustworthy, elegant

2. **Bold Playful**: #E91E8C (magenta) + #00D9FF (cyan) + #FFFFFF (text)
   - Used by: Entertainment, youth content, viral carousels
   - Mood: Energetic, fun, attention-grabbing

3. **Warm Organic**: #C65911 (terracotta) + #D4AF37 (gold) + #F5EEE0 (cream)
   - Used by: Lifestyle, wellness, natural products
   - Mood: Warm, welcoming, authentic

4. **Clean Editorial**: #001F3F (navy) + #FFFFFF (white)
   - Used by: News, how-to, education
   - Mood: Professional, clear, trustworthy

**Font Pairings (verified from pins):**
- Headline: Poppins Bold (700-900)
- Body: Inter Regular (400)
- Accent: Playfair Display (scripts/elegant headlines)
- Alternative: Montserrat (bold, modern)

**Layout Patterns:**
1. **Hero Image Left + Text Right (40/60 split)**
   - Image quality: High-res, lifestyle photography
   - Text alignment: Left or center
   - Safe zone: 20% margins all sides
   - Use: Product features, tips, transformations

2. **Full-Bleed Image + Centered Text Overlay**
   - Background: Photograph or gradient
   - Text overlay: Semi-transparent dark bg (rgba 0,0,0,0.5)
   - Text color: White (#FFFFFF) for contrast
   - Font size: 36-48px headline, 16-20px body
   - Use: Inspirational quotes, hooks, CTAs

3. **Grid Layout (3x3 or 2x2)**
   - Element size: ~100x100px per item (3x3) or 200x250px (2x2)
   - Spacing: 16-20px between items
   - Use: Checklists, tips, benefits, comparisons

**Visual Elements:**
- Rounded corners: 12-16px on images
- Shadows: Subtle (box-shadow: 0 2px 8px rgba)
- Icons: Outline style (2-3px stroke), 24-32px
- Geometric shapes: Circles, squares, triangles for hierarchy

## Copy Strategy Patterns

### Hook Formulas (First Slide = Stop Scroll)

**By Emotion:**

**Fear/Urgency:**
- "STOP. You're probably making this mistake..."
- "⚠️ Before you [action], know this..."
- "Most people get this wrong (including you)"

**Hope/Aspiration:**
- "What if [outcome] was actually achievable?"
- "✨ [X people] just discovered how to [result]..."
- "The secret to [goal]: [teaser]..."

**Curiosity/Mystery:**
- "👀 This [industry] hack nobody talks about..."
- "❓ What [audience] gets wrong about [topic]"
- "This is so [adjective], even [expert] doesn't know..."

**Joy/Entertainment:**
- "😂 POV: You just realized [relatable thing]"
- "Wait till you see this... (you're gonna love it)"
- "This is so [adjective] I can't even..."

**Anger/Truth:**
- "🔥 They don't want you to know this..."
- "STOP. They've been LYING about [topic]..."
- "This [industry] SECRET is exposed..."

### Value Slide Progression

**Slide 2-3 (Curiosity Build):**
- Tease solution
- Create tension or contrast
- Myth-busting setup
- "But here's the twist..."

**Slide 4-7 (Value Delivery - Pick 1-3):**
- Listicle format: "Mistake #1", "Tip #2"
- Step-by-step: "Step 1:", "Step 2:"
- Before-after: Show problem → show solution
- Story arc: Relatable failure → breakthrough
- Each slide must be implementable TODAY

**Slide 8-9 (Proof):**
- Social proof: "X thousand people already..."
- Testimonial: Real person name + result
- Stat: "87% of users saw [result]"
- Case study: Specific transformation
- Make promise BELIEVABLE

**Slide 10+ (CTA):**
- Clear ask: Follow / Link / DM / Save
- Benefit: "For more tips" / "Get access"
- Urgency: "Last 3 spots" / "Closes Friday"
- One-click friction

### Copy Techniques (Most Effective)

1. **Listicle**: "10 Mistakes" → High engagement, clear structure
2. **Before-After**: Show problem visually → Solution → Result
3. **Story**: Relatable struggle → Turning point → Resolution
4. **How-To**: Step-by-step implementation → Quick wins
5. **Myth-Busting**: Common belief → Truth reveal → Why it matters
6. **Comparison**: X vs. Y → Reality check → Which wins
7. **Urgency**: Limited time / Limited spots / Closing date
8. **Curiosity Loop**: Teaser → Build tension → Reveal → CTA

## Slide Count Decision Matrix

| Topic | Narrative | Optimal Slides | Why |
|-------|-----------|---|---|
| Listicle | Hook → 8 mistakes → Proof → CTA | 10 | Needs all mistakes + credibility |
| Tutorial | Hook → 3 steps → Proof → CTA | 6-7 | Too many steps = lost audience |
| Before-After | Hook → Problem → Solution → Results → CTA | 5-6 | Simple arc, doesn't need complexity |
| Myth-Busting | Hook → Myth → Truth → Proof → CTA | 6 | Clear structure, balanced |
| Story | Hook → Struggle → Turning point → Resolution → CTA | 7 | Narrative needs breathing room |
| Product | Hook → Feature 1 → Feature 2 → Social proof → CTA | 5-6 | Features bore fast, proof sells |

**Rule:** If story doesn't need slide → DELETE. Retention drops 10-15% per weak slide.

### Phase 3: Copy Strategy Analysis (15 pins, 10 min each)
**Goal:** Rank copy techniques by effectiveness

Analyze per pin:
- Hook type (fear/hope/curiosity/joy/anger)
- Technique (listicle/story/before-after/how-to/myth)
- Slide count (3/5/7/10)
- Estimated retention (would you swipe all?)

**Output:** Technique ranking
```
Listicle: 94% avg engagement
Before-After: 89% avg engagement
Story: 85% avg engagement
How-To: 83% avg engagement
```

### Phase 4: Layout + Element Mapping (20 pins total)
**Goal:** Identify dominant layout patterns

Analyze:
- Background style (image/gradient/solid)
- Text placement (overlay/side/bottom)
- Text alignment (left/center/right)
- Visual elements (icons/graphics/mockups)
- Rounded corners (yes/no, size)
- Shadows (yes/no, depth)

**Output:** Layout distribution
```
Hero image left + text right: 38%
Full-bleed image + overlay text: 31%
Grid layout: 18%
Asymmetrical + whitespace: 13%
```

### Phase 5: Cross-Pin Coherence (50 pins total)
**Goal:** Understand feed narrative + brand identity

Analyze:
- Do posts relate (topic, emotion, design)?
- Is there narrative flow (series feel)?
- Are fonts/colors consistent?
- Does tone match audience?

**Output:** Coherence guidelines
```
Top brands use 2-3 fonts consistently
Top brands use 3-5 colors locked
Posts relate via topic (not scattered)
Emotional tone stays 1-2 primary emotions
```

## FeedIA Rules Generated

After analyzing 50 pins:

### 1. Typography Rules
```
IF topic = education/professional
  THEN headline = Poppins OR Montserrat
  THEN body = Inter OR Open Sans
ELSE IF topic = luxury/premium
  THEN headline = Playfair Display
  THEN body = Lora
ELSE (default/playful)
  THEN headline = Poppins
  THEN body = Inter
```

### 2. Color Rules
```
IF emotion = fear/urgency
  THEN palette = Dark Premium OR Bold Playful
IF emotion = hope/aspiration
  THEN palette = Warm Organic OR Clean Editorial
IF emotion = curiosity/entertainment
  THEN palette = Bold Playful OR Warm Organic
```

### 3. Layout Rules
```
IF slide = 1 (hook)
  THEN background = hero image full-bleed OR gradient
  THEN text placement = overlay centered
IF slide = 2-7 (value)
  THEN background = image OR subtle pattern
  THEN text placement = side OR overlay
IF slide = 8+ (CTA)
  THEN background = solid accent color OR gradient
  THEN text placement = centered
```

### 4. Copy Rules
```
IF audience = B2B
  THEN use = How-To OR Case Study (conservative)
IF audience = lifestyle/entertainment
  THEN use = Story OR Before-After OR Myth-Busting (emotional)
IF goal = viral
  THEN use = Listicle OR Curiosity Loop (high engagement)
IF slide count = ?
  THEN optimal = 7 (sweet spot)
  THEN avoid = 10+ (retention drops)
```

### 5. Retention Rules
```
EVERY slide must have = retention trigger (why swipe next?)
IF no retention trigger = delete slide
IF weak hook = <50% reach slide 5
IF strong narrative arc = 75%+ reach end
IF 10 slides weak = 30-40% retention
IF 7 slides strong = 75-85% retention
```
