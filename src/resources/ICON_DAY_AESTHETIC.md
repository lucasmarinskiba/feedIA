## TYPOGRAPHY SPECIFICATIONS

### Headline (Primary)
```
Font: Bold sans-serif (Montserrat, Poppins, Inter)
Size: 80-130px
Weight: 700-900 (Bold/Extra Bold)
Color: White or gold
Positioning: Overlay left-center or diagonal
Spacing: Tight letter-spacing (2-5px)
Shadow: Subtle drop shadow or glow effect
Examples: "THE ICON DAY", "UNSTOPPABLE", "LEGEND IN MAKING"
```

### Subject Name (Secondary)
```
Font: Bold sans-serif (matching headline)
Size: 40-60px
Weight: 600-700
Color: White or gold
Positioning: Signature placement (usually lower right)
Spacing: Natural letter-spacing
Style: Elegant signature or clean block lettering
```

### Supporting Details (Tertiary)
```
Font: Clean sans-serif
Size: 18-28px
Color: White or gold
Positioning: Corners or discretion placement
Content: Country, year, achievement details
```
