# Proposal Generation Agent: Pitch System

**AI-powered pitch deck + written proposal for sales closing.**

---

## PROPOSAL ARCHITECTURE

| Section | Content | Length | Goal |
|---------|---------|--------|------|
| Hook (Slide 1-2) | Problem statement + emotional resonance | 30 sec | Capture attention |
| Authority (Slide 3-5) | Case studies + proof of concept | 2 min | Build credibility |
| Solution (Slide 6-8) | Product/service + unique positioning | 2 min | Show differentiation |
| Value (Slide 9-11) | ROI + benefits + pricing | 1.5 min | Justify investment |
| Closing (Slide 12) | CTA + next steps + contact | 30 sec | Conversion |

---

## COPY TEMPLATES BY SECTION

### Hook (Problem + Opportunity)
```
"Your [PROBLEM] costs [COMPANY] ~$[X]/month in lost [METRIC].
Most [INDUSTRY] see [STAT]% improvement after [SOLUTION].
Here's how [BRAND] delivered [RESULT] in [TIMEFRAME]."
```

### Authority (Proof)
```
"Case Study: [CLIENT] faced [PROBLEM].
We implemented [SOLUTION].
Result: [METRIC] improved [X]% in [TIMEFRAME].
Cost: [ROI_MULTIPLE]X return."
```

### Solution (Differentiation)
```
"Unlike [COMPETITOR_APPROACH], we [UNIQUE_ANGLE].
This means [BENEFIT_1], [BENEFIT_2], [BENEFIT_3].
Reason: [TECHNICAL_MOAT]."
```

### Value (Investment)
```
"Investment: $[X] over [TERM].
ROI: $[Y] recovered in [TIMEFRAME].
NPV: $[Z] at current growth rate.
Bonus: [RISK_MITIGATION]."
```

## DESIGN SPECS FOR PITCH DECK

| Slide | Visual | Copy | Mood |
|-------|--------|------|------|
| 1-2 | Full-bleed image + centered text | Hook problem | Urgent confident |
| 3-5 | Case study grid (3x2) + metrics | Proof testimonial | Authoritative |
| 6-8 | Product showcase + feature breakdown | Solution explanation | Professional |
| 9-11 | ROI calculator visual + pricing table | Value financial | Trustworthy |
| 12 | Brand icon + CTA button | Next steps | Action-oriented |
