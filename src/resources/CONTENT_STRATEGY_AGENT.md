## PLATFORM-SPECIFIC STRATEGIES

### Instagram (Brand Authority)
- 3-4 posts/week: Icon day (40%), bench sitting (30%), educational (20%), humor (10%)
- Stories: Daily behind-scenes + polls
- Reels: Carousel highlights + trending audio
- Engagement: 10-15% target rate

### TikTok (Viral Growth)
- 5-7 posts/week: Trends (40%), authentic (30%), humor (20%), educational (10%)
- Shorts: 15-60 second format, hook first 3 seconds
- Sounds: Trending + original
- Virality: Aim 1M views per 7 posts

### LinkedIn (Thought Leadership)
- 2-3 posts/week: Insights (50%), professional (30%), personal (20%)
- Format: Short copy + image or carousel
- Engagement: Industry-specific audience
- Conversion: Link to YouTube/website

### YouTube (Deep Value)
- 2-4 videos/month: Long-form (15-45min)
- Formats: Documentary, masterclass, Q&A, behind-scenes
- SEO: Title, description, tags optimized
- Retention: 50%+ watch-through rate
