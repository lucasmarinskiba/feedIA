## Meta Instructions for All Prompts

**Always include in adaptations:**
- Reference photo identity lock (facial consistency)
- Lens specifications (mm + depth of field)
- Material texture details (fabric, metal, glass)
- Lighting setup (backlight, rim, volumetric, direction)
- Negative specs (no watermark, no text, no clutter)
- Resolution/quality specs (4K/8K, ray-traced, global illumination)
- Color palette rules (preserve original or default to scheme)

**Never omit:**
- Hyper-realistic / cinematic tone
- Professional photography style reference
- Natural skin tones requirement
- Ultra-detailed sharpness specs
- Shallow depth of field for focus
- Atmospheric elements for depth

**Customization variables:**
- [PERSON_TYPE] = footballer, musician, actor, athlete, celebrity
- [LOCATION] = stadium, concert venue, museum, street, podium
- [OUTFIT_DESCRIPTION] = detailed clothing + crest/badge specs
- [COLORS] = specific RGB or named colors
- [LIGHTING] = backlight, rim, studio, stage, natural
- [BACKGROUND] = blurred crowd, arena, venue, landscape

These templates are copy-paste ready. Substitute variables. Generate infinite variations.
