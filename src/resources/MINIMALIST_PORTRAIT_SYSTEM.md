## TYPOGRAPHY SPECIFICATIONS

### Headline Type
- Font: Bold sans-serif (Montserrat, Inter, Poppins)
- Weight: 700-900
- Size: 72-120px
- Color: White or gold
- Positioning: Overlay left/center/diagonal
- Spacing: Tight letter-spacing
- Shadow: Subtle drop or gold glow

### Subject Name Type
- Font: Bold sans-serif
- Weight: 600-700
- Size: 36-48px
- Color: White, gold, or primary accent
- Positioning: Bottom-right or signature placement

### Supporting Text
- Font: Clean sans-serif
- Size: 18-24px
- Color: White or gold
- Positioning: Corners, discreet placement
