## SILHOUETTE BACKGROUND SPECIFICATIONS

### Silhouette Positioning
```
Subject positioned from behind, larger/taller than foreground.
Creates layering illusion of person behind subject.
Semi-transparent, creates ghosted background effect.
Higher opacity (40-60%) for visible but ethereal effect.
```

Name "[PLAYER_NAME]" integrated into silhouette back area.

## DOUBLE EXPOSURE LAYERING TECHNIQUE

### Layer 1: Foreground Subject
```
Sharp focus on foreground pose and expression.
Subject: Clear crisp details, sharp edges.
Lighting: Bright cinematic lighting emphasizing detail.
Opacity: 100% foreground, naturally opaque.
```

### Layer 2: Background Silhouette
```
Soft focus silhouette behind foreground.
Large scale: Taller/wider than foreground for layering.
Opacity: 40-60% for visible but ethereal effect.
Color: Darker shade creating silhouette contrast.
```

### Layer 3: Text Integration
```
Name and number integrated into silhouette.
Semi-transparent text, readable but not dominating.
Color: Slightly darker than silhouette for contrast.
Positioning: Center back area, follows silhouette.
```

### Blending Mode: Screen or Overlay
```
Creates layered ghosting effect, double exposure aesthetic.
Maintains foreground crispness while showing background.
Creates achievement symbolism: present and historical layers.
Professional double exposure result.
```
