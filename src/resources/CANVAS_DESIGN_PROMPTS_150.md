## IMPLEMENTATION GUIDE

### 1. Template-Based Generation
```
Load Template → Extract Structure
→ Fill with [TOPIC] content → Apply [BRAND_KIT] colors → Export
```

### 2. Canvas Automation
```
Import [USER_BRIEF]
→ Select [TEMPLATE_CATEGORY] (carousel/post/video/etc)
→ Apply [THEME] (dark/light/vibrant/minimal)
→ Auto-fill [VARIABLES] (text, images, colors)
→ Generate [VARIATIONS] (3 color schemes, 2 layouts)
→ Output → Canva link + PNG/MP4
```

### 3. Design System
```
Brand Kit
├── Colors (primary, secondary, accent)
├── Fonts (heading, body, accent)
├── Logos (vertical, horizontal, icon)
├── Templates (carousel, post, video, reel)
└── Elements (icons, shapes, frames)
```

### 4. Multi-Platform Export
```
Design Once → Export 5 Formats
├── Instagram: 1080×1350 + 1080×1920 story
├── TikTok: 1080×1920 vertical
├── YouTube: 1920×1080 horizontal
├── Pinterest: 1000×1500 portrait
└── LinkedIn: 1200×627 feed
```

## FEEDIA CANVAS INTEGRATION

```typescript
async function generateDesignInCanva(brief: DesignBrief): Promise<CanvaOutput> {
  // 1. Load Canvas template library
  const templates = await loadCanvaTemplates(brief.type);  // carousel/post/video
  
  // 2. Select template + theme
  const template = templates.find(t => t.matches(brief));
  const themed = applyTheme(template, brief.theme);
  
  // 3. Fill content
  const populated = fillContent(themed, {
    headline: brief.text,
    images: brief.images,
    brand: brief.brandKit
  });
  
  // 4. Generate variations
  const variations = generateVariations(populated, 3);  // 3 color schemes
  
  // 5. Export all formats
  const outputs = await exportAllFormats(variations);
  // Instagram + TikTok + LinkedIn + Pinterest + YouTube
  
  return outputs;
}
```
