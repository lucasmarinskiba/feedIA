## MASHUP ANGLES (8 Combination Methods)

### Angle 1: INVERSION (Opposite + Contrast)
```
Take pattern + flip core assumption
Example: Trophy vitrina + object metaphor = "Trash Can as Life Achievement Vitrina"
Subject gazes at pile of trash as if it's trophy. Humorous deconstruction.
```

### Angle 2: SCALE SHIFT (Tiny → Huge or vice versa)
```
Miniature + macro scale contrast
Example: Miniature doll + hyperrealistic detail = "Doll wearing embroidered FIFA crest with 8K detail"
Toy becomes ultra-realistic through detail obsession.
```

### Angle 3: CATEGORY COLLISION (Opposite worlds)
```
Combine two unrelated categories
Example: Pet FIFA + Luxury jewelry = "Dog wearing championship diamond pendant"
Gaming lounge + trophy vitrina = "Gaming setup shrine to trophies"
```

### Angle 4: EMOTIONAL INVERSION (Reverse feeling)
```
Positive + negative emotion clash
Example: Icon day pose + crying expression = "Legendary player overwhelmed by weight of legacy"
Championship celebration + grief = "Victory as loss mourning"
```

### Angle 5: TEMPORAL MASH (Past + Present + Future)
```
Combine timeline layers (what double exposure does naturally)
Example: Young street footballer + current champion + elderly coach = "Achievement lifecycle in one frame"
```

### Angle 6: CONTEXT TRANSPLANT (Wrong environment)
```
Move concept to incongruous setting
Example: Trophy vitrina + shopping mall = "Achievement sold between clothing stores"
Bench sitting + luxury penthouse = "Athlete contemplating on gold bench"
```

### Angle 7: TOOL/OBJECT SUBSTITUTION (Replace prop)
```
Swap trophy for unexpected object
Example: Trophy vitrina but with: food, fashion, books, furniture, weapons, art
Subject gazes longingly at championship plate of food, or hat, or novel
```

### Angle 8: MULTI-SUBJECT INTERACTION (Add characters)
```
Two+ subjects in unexpected relationship
Example: Athlete + pet + mascot all in double exposure pose
Celebrity + historical figure + fictional character in same FIFA card
Miniature doll + life-size person in same trophy showcase
```
