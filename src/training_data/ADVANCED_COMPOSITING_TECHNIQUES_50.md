## VISUAL TECHNIQUES EXTRACTED

### Technique 1: Brand Swirl Effect
```
Liquid/smoke dynamic flowing around text + subject
Swirling motion envelope wrapping brand name
Blended with subject body/clothing
Creates: fluid brand integration, movement, premium feeling
Color: single saturated accent (purple, electric blue, magenta)
```

### Technique 2: Clean Product Shot + Color Block
```
Subject isolated on single saturated background color
Product/logo visible but small (corner or side)
Minimal composition, maximum focus
Studio-quality lighting, no distractions
Creates: authority, product clarity, professional
```

### Technique 3: Neon Glow Overhead Extreme
```
Glow effects on face/limbs (neon outline)
Shot from directly above (extreme overhead angle)
Subject mid-motion (jumping, dynamic pose)
High contrast neon + muted background
Creates: futuristic, energetic, technical
```

### Technique 4: Extreme Closeup Selfie Distortion
```
Ultra-wide angle lens on face (very close)
Exaggerated proportions (nose/eyes larger)
Studio neutral backdrop
Intimate framing fills entire frame
Creates: charisma, relatability, personality amplified
```

### Technique 5: Fisheye Ground-Level + Sky
```
Fisheye lens distortion from ground angle looking up
Bright sky background (blue, clear, overexposed)
Subject heroic pose mid-action
Product visible (not central) in frame
Creates: hero shot, power amplified, ground-to-sky perspective
```
