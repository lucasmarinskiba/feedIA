## VISUAL TECHNIQUES EXTRACTED

### Technique 1: Phone/Rectangle Covers Face + Large Typography Overlay + Accent Box
```
Subject wearing hood/neutral outfit, phone or rectangle shape covering face
Large serif or bold typography overlaid on entire composition
Single accent color box highlighting key message
Minimalist white/neutral background
Creates: mystery, intrigue, modern editorial feel
Focus on typography as primary visual element
```

### Technique 2: B&W Photo + Neon Highlight Sticker Effect + White Border Cutout
```
Portrait in black and white, desaturated
Neon single-color highlights (green, pink, yellow) on clothing/accessories
White sticker-like border cutout effect on subject
Urban or grayscale background
Typography scattered in neon color throughout
Creates: modern digital aesthetic, contrast, high energy
```

### Technique 3: Person + Floating Social Post Card + Neon Glow Background
```
Subject in workspace (laptop, professional setting)
Floating social media post card above/beside subject
Neon glow effect in background (bokeh, lights)
City skyline or blurred nighttime background
Creates: social authority, professional positioning, modern
Post card is secondary focal point, subject is primary
```

### Technique 4: Giant Product Scaled + Person Small + Solid Background + Stacked Typography
```
Oversized product (shoe, object, item) dominates frame
Person sitting/positioned small in relation to product
Solid single-color background (pink, red, blue)
Typography stacked/layered in multiple colors
Product becomes background element, person focal point
Creates: product storytelling, personality-driven, playful scale
```

### Technique 5: Multiple Faces + Inverted Orientation + Solid Color + Typography Scatter
```
Two or more faces/subjects at different angles
One or more faces inverted/upside down
Solid vibrant color background (red, pink, orange)
Typography scattered playfully around faces
Energetic chaotic composition
Creates: movement, playfulness, high energy, engagement
Faces convey emotion/expression prominently
```

## ADAPTATION TEMPLATE

```
[PERSON/PEOPLE]
Technique: [PRIMARY DESIGN TECHNIQUE]
Product/Object: [IF APPLICABLE]
Primary Color: [MAIN BACKGROUND/ACCENT COLOR]
Secondary Colors: [SUPPORTING COLORS]
Typography: [SERIF/SANS-SERIF/PLAYFUL]
Layout: [MINIMALIST/CHAOTIC/BALANCED]
Vibe: [EDITORIAL/PLAYFUL/AUTHORITY/MODERN]
Background: [SOLID/BOKEH/URBAN/BLURRED]
Emotional Tone: [INTRIGUE/ENERGY/AUTHORITY/JOY]
```

**Variables:**
- [PERSON/PEOPLE] = model, influencer, creator, professional, personality
- [PRODUCT] = shoe, object, item, brand element, visual device
- [COLOR] = pink, red, green, blue, yellow, orange, purple
- [BACKGROUND COLOR] = matching solid palette color
- [OUTFIT] = hood, casual, professional, sporty, neutral
- [SOCIAL PLATFORM] = Instagram, TikTok, LinkedIn, Twitter
- [HEADLINE] = brand message, call-to-action, motivational quote
- [EMOTION] = joy, intrigue, confidence, energy, mystery
