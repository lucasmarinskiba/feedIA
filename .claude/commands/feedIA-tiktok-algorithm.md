# FeedIA · Algoritmo & Dinámica TikTok

Modelo del FYP de TikTok y cómo ganarle. NO Instagram: acá manda el **interés puro**
(completion + watch time + rewatch + shares + replays), casi sin peso de followers.

## SEÑALES QUE MUEVEN EL FYP (orden de impacto)

1. **Completion rate** (vio el video entero) — reina.
2. **Rewatch / replays** (loop) — multiplica.
3. **Watch time promedio** (segundos vistos).
4. **Shares + sends por DM** > saves > comments > likes.
5. **Velocidad** (engagement en primeros minutos/horas → test batch inicial).
6. **Sonido trending** (audio en alza = empuje extra).
7. **Re-views del mismo user** (volvió a tu perfil).

## CÓMO LLEGAR AL PÚBLICO CORRECTO

- TikTok testea cada video en mini-lotes; si performa, expande. → el **hook 0-2s + completion** del primer lote define todo.
- Nicho claro y consistente → TikTok aprende a quién mostrarte.
- Caption + on-screen + sonido + hashtags = señales de tema (no spamear hashtags).
- Responder comentarios con video (reply) = más alcance.

## PALANCAS ACCIONABLES

| Objetivo       | Palanca                                                           |
| -------------- | ----------------------------------------------------------------- |
| Más alcance    | subir completion: video más corto que el guion, loop, hook brutal |
| Salir del pozo | 3-5 videos/semana mismo nicho, 7-14 días, sin borrar              |
| Más watch time | cliffhangers por beat, ritmo, pattern interrupts                  |
| Empuje extra   | trending sound + format del momento                               |
| Mejor timing   | publicar cuando tu audiencia está activa (analytics cuenta)       |

## DIAGNÓSTICO (si baja alcance)

- ¿Completion < 50%? → hook/ritmo/duración.
- ¿Borraste videos / cambiaste de nicho? → reset de aprendizaje.
- ¿Watermark de otra app (CapCut/IG)? → penaliza; exportar limpio.
- ¿Texto/baneable? → revisar `/feedIA-tiktok` compliance.
