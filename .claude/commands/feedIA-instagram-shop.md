# FeedIA · Tienda Instagram (IG Shopping)

Monta y optimiza la tienda nativa de Instagram para que el contenido sea comprable.

## Setup

- **Commerce Manager + catálogo Meta** (manual, e-commerce platform o feed).
- **Requisitos**: cuenta business, cumplir políticas de comercio, dominio verificado.
- **Pestaña Shop** + **colecciones** temáticas curadas.

## Operación

- **Product tags** en posts, Reels y Stories (sticker de producto).
- **Fichas**: foto principal limpia, título con beneficio, precio, variantes.
- **Checkout** (donde esté disponible) o link a sitio.
- Contenido que etiqueta producto sin romper estética del feed.

## Métricas

Vistas de producto, tags clic, add-to-cart, conversión, ROAS si hay ads.
