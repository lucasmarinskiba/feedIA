Live Shopping = IG Live + product tags → ventas en vivo + replay con tags persisten.

## Según $ARGUMENTS

**"plan [oferta]"** → Plan completo de live shopping: pre-promo, script, post-live.

**"script [duration]"** → Script estructurado: hook + demo + Q&A + scarcity + CTA.

**"collab-live [@partner]"** → Live con partner para 2x audience.

**"pre-promo"** → 7-day pre-promo calendar (stories + broadcasts + posts).

**"replay-optimize"** → Cómo optimizar replay para conversiones post-live.

**"metrics [live-id]"** → Métricas: peak viewers, avg watch, taps, conversions.

## Estructura Live Shopping (60-90 min óptimo)

```
0-5 min:   Welcome + hook. "Quien quede los 90 min se lleva regalo X"
5-15 min:  Storytelling — por qué este producto, qué problema resuelve
15-35 min: Demo en vivo del producto, casos de uso
35-50 min: Q&A audiencia (preguntas pre-cargadas + chat live)
50-65 min: Testimonios reales (clips o invitados)
65-80 min: Oferta exclusiva live (precio mejor que web, tiempo limitado)
80-90 min: Cierre + recordatorio scarcity + reply DMs
```

## Pre-promo (7 días antes)

| Día | Acción                                                     |
| --- | ---------------------------------------------------------- |
| -7  | Anuncio en feed: "Live shopping martes 8pm. Exclusivo."    |
| -5  | Story con countdown sticker                                |
| -3  | Reel teaser "lo que voy a mostrar"                         |
| -2  | Broadcast channel: "no te lo pierdas, oferta solo en vivo" |
| -1  | Story con poll "¿estás listo?" + reminder                  |
| -0  | Story 1h antes "EN VIVO en 1h" + 15 min antes              |

## Conversion mechanics

### Scarcity real (no fake)

- "Solo 20 unidades a este precio"
- "Oferta válida hasta el final del live + 1 hora"
- "Primeros 10 reciben bonus extra"

### Urgency

- Timer visible si plataforma lo permite
- "Quedan X minutos de live"
- Stock real-time updates

### Social proof live

- Mencionar nombres de quienes compraron en vivo
- Mostrar comentarios de fans
- Testimonios en chat

## Métricas live shopping

| Métrica                 | Benchmark           |
| ----------------------- | ------------------- |
| Peak concurrent viewers | 5-10% de followers  |
| Avg watch time          | 12-25 min           |
| Product tap rate        | 8-18%               |
| Add to cart             | 10-20% de taps      |
| Conversion live         | 3-8%                |
| Conversion replay 48h   | 1-3% (sumar a live) |
| Q&A engagement          | 25%+ comentarios    |

## Replay optimization

Post-live (en primeras 2h):

- ✅ Edit highlights del replay (3-5 min)
- ✅ Stories con clips clave
- ✅ Reel con momento viral del live
- ✅ Carrusel con FAQ respondidas
- ✅ Subir replay como IGTV con tags persistentes
- ✅ DM a peak viewers que no compraron

## Collab Live (Doble alcance)

- 2 creadores → audiences sumadas
- 1 host + 1 invitado expert
- Brand + influencer launch

Tasa conversion 1.5-2x vs solo live.

## Anti-patterns

- ❌ Live < 30 min → no construye momentum
- ❌ Sin pre-promo → 5 viewers
- ❌ Solo vender, no educar → audience hostil
- ❌ Sin Q&A → unidireccional, baja retention
- ❌ Sin replay edit → pierde 60% del valor
