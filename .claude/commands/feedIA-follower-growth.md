# FeedIA · Crecimiento de Seguidores

Seguir es una decisión: hay que dar alcance frío Y una razón clara para apretar "seguir".

## Palancas

1. **Alcance frío**: Reels (IG) / FYP (TikTok) que llegan a no-seguidores. Volumen + hooks.
2. **Razón para seguir**: promesa de cuenta clara ("acá aprendés X cada día") + series.
3. **Perfil que convierte**: foto, @ claro, bio con propuesta + para quién + CTA, destacadas/pinned.
4. **Loop de exposición**: saves/shares (IG) y rewatch/shares (TikTok) exponen a nuevos.
5. **Colabs**: collab posts (IG) y duet/stitch (TikTok) toman audiencia ajena.
6. **Consistencia**: cadencia alta y constante > ráfagas.

## Diagnóstico

Si hay alcance pero no follows → problema de perfil/promesa. Si no hay alcance → hook/formato.
