Pricing skill para FeedIA. Genera estrategias de precio + tiers + Van Westendorp + value metrics.

## Según $ARGUMENTS

**"tiers [producto]"** → 3 tiers (Starter/Pro/Business) con features matrix + price points.

**"freemium"** → Cuándo conviene + límites del free + upgrade triggers.

**"trial [días]"** → Trial strategy: 7/14/30 days, credit card upfront?, conversion tactics.

**"van-westendorp [survey]"** → Procesa datos survey → curva de precio óptimo.

**"value-metric"** → Identifica qué métrica usar como base de precio (users/orders/data/seats).

**"increase [%]"** → Plan para subir precio sin churn: grandfather, communication, timing.

**"anchoring"** → Estructura para anclar el precio percibido (decoy effect).

**"comp [competidores]"** → Tabla comparativa de precios competencia + posición sugerida.

## Frameworks

### Tiers structure (regla 1-2-3)

- **Tier 1 (Starter)**: $X — para validación, 10-15% conversión a Pro
- **Tier 2 (Pro)**: $2-3X — sweet spot, 60-70% de revenue
- **Tier 3 (Business)**: $5-10X — high-touch, custom features

### Value metrics ranking

1. **Seats/users** — escala con tamaño equipo (SaaS)
2. **Usage** — API calls, GB, requests (consumption)
3. **Outcomes** — leads generated, sales (results-based)
4. **Tiers** — flat con features differentiation (más común)

### Decoy effect

```
Plan A: $30/mo - 50 features
Plan B: $50/mo - 100 features (ANCHOR, raramente comprado)
Plan C: $45/mo - 100 features + soporte (TARGET)
```
