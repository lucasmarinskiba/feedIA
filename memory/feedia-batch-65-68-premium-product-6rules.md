# 6-Rule Framework (Embedded in Every Prompt)

**Rule 1 (Cover)**: Max 8 words. Promise/number. Big font, contrast. Stop scroll.
**Rule 2 (Slide 2)**: Problem setup + stakes. Hook confirmation.
**Rule 3 (Standalone)**: Every slide works as screenshot alone. Max 2 fonts, high contrast.
**Rule 4 (Save-Worthy)**: Framework/prompt/quotable. Share-worthy core slide.
**Rule 5 (AI Scaffold)**: AI builds structure (topic/angle/brand). Humans edit taste.
**Rule 6 (Caption)**: Restate idea + add context + comment trigger + freebie. One carousel = 3 wins.
