# Product Swap Photography Templates

**Core Technique**: Image A (scene reference) + Image B (replacement garment) → Replace ONLY the garment in Image A with garment from Image B while preserving 100% of scene elements, lighting, positioning, fabric behavior, shadows.

**Critical Rules**:
- ✓ ONLY garment changes
- ✓ Camera angle identical
- ✓ Composition exact same
- ✓ Lighting direction same
- ✓ Shadows preserved
- ✓ Hanger/rod/transparent elements unchanged
- ✓ Gray background same tone
- ✓ Depth of field identical
- ✓ Fabric drape/folds natural matching original garment weight
- ✗ Do NOT alter: camera position, framing, crop, perspective, distance, lighting, background, reflections, overall studio mood
